How To Make Language

by Ryan McDevitt

HTML stands for HyperText Markup Language
The most basic form,
Creating a website is easy,
You could do it in your dorm


There are so many HTML tags,
Like beta, title, and head,
They are highlighted in colors,
Like the color blue, or red


em, p, and strong are HTML tags
Just to name a few more,
If you have been dying to start a website,
This is definitely your cure.
